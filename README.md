# SPEKTOR. — лендинг на Vue 3 + Tailwind CSS

Дизайн "Dark Glass Neon": стеклянные карточки, неоновый оранжевый акцент,
3D-орб на Three.js в хиро, скролл-анимации на GSAP, переключатель RU/UZ и
переключатель светлой/тёмной темы.

## Стек

- Vue 3 (`<script setup>`, Composition API)
- Vite
- Tailwind CSS (подключён, используется вместе с готовой дизайн-системой в `src/style.css`)
- GSAP + ScrollTrigger — скролл-анимации и счётчики цифр
- Three.js — 3D-орб в хиро-секции

## Установка и запуск

Нужен установленный Node.js версии 18 или новее.

```bash
npm install
npm run dev
```

Откроется `http://localhost:5173` — сайт с горячей перезагрузкой.

## Сборка для продакшена

```bash
npm run build
```

Готовые файлы появятся в папке `dist/` — их можно залить на любой хостинг
(Vercel, Netlify, обычный хостинг с раздачей статики и т.д.).

Проверить собранную версию локально:

```bash
npm run preview
```

## Что уже настроено

- **RU/UZ** — весь текст сайта берётся из `src/i18n.js`, переключение мгновенное,
  без перезагрузки страницы.
- **Тёмная/светлая тема** — кнопка-лампочка в шапке, выбор запоминается в
  `localStorage`, при следующем визите подставляется без "мигания" (скрипт в
  `index.html` выставляет тему до отрисовки Vue-приложения).
- **Анимации** — при первом появлении секции в зоне видимости она плавно
  проявляется снизу (`v-reveal`), карточки в сетках — с небольшой задержкой
  друг за другом (`v-reveal-group`). Цифры в мини-дашборде хиро считаются от
  0 до целевого значения (`v-countup`). При `prefers-reduced-motion` анимации
  отключаются автоматически.
- **Форма квалификации лида** — открывается по клику на "BOSMANG! / НЕ
  НАЖИМАТЬ!", закрывается по крестику, клику на фон или Esc. Сейчас submit
  ничего никуда не отправляет (только эмулирует успех в интерфейсе) — нужно
  подключить к вашему бэкенду/CRM (см. `handleSubmit` в `QualifyModal.vue`).

## Что стоит доделать перед продакшеном

- Подключить реальную отправку формы (fetch/axios на ваш API, Bitrix24,
  amoCRM и т.д.) в `src/components/QualifyModal.vue`.
- Заменить плейсхолдеры кейса и отзывов на реальные данные в `src/i18n.js`
  (ключи `caseH2`, `caseP`, `rev1`…`rev3` и т.д.).
- Добавить свой favicon и OpenGraph-превью в `index.html`.
